# Kolv2Backend

Kolv2Backend is an API created for the Kolveniershof project.

## Installation

1. Download it
2. Open editor
3. Run "npm i"
4. Run "npm start" or "npm start-local"
5. Browse to "localhost:3000"

This API can also be viewed online on: https://kolv02-backend.herokuapp.com/
